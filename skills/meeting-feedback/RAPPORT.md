# Rapport template (French)

The report is always produced in **French**, with **exactly** these sections, in this
order, addressing the person as **tu**. The six scorecard axes never change — that is
what makes reports comparable from one meeting to the next.

```md

## 📋 Contexte

- **Réunion** : <sujet, durée si connue>
- **Participants** : <liste, avec rôle si identifiable>
- **Rôle de <Personne>** : <animateur / contributeur / observateur…>
- **Temps de parole estimé** : <part approximative, plus long monologue>

> **Fiabilité de l'attribution** : <"Aucun doute" OU liste courte des segments exclus
```

## Style rules

- **Tutoiement**, direct, zéro langue de bois — mais chaque jugement est adossé à une
  citation verbatim du transcript.
- Pas de sandwich à compliments : les sections font le tri, pas l'enrobage.
- Les notes de la scorecard sont exigeantes : 5/10 = prestation moyenne ; 8+ se mérite.
- Une action du plan d'action doit être **vérifiable dans le prochain transcript**
  (« sois plus confiant » est interdit ; « termine tes propositions par une question de
  décision : "on part là-dessus ?" » est le standard).
