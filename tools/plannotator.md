# Plannotator

- **Link:** https://plannotator.ai/
- **Credit:** [backnotprop](https://github.com/backnotprop/plannotator)

## What it is

A browser UI for reviewing AI output: annotate plans, and review code or markdown visually. It returns your annotations to the agent so it can revise.

## Why I use it

Tightens the plan-review loop — I mark up plans and diffs visually instead of a long back-and-forth in chat. The feedback comes back structured and actionable.

## Install

Also available as a Claude Code plugin from the `plannotator` marketplace:

```
/plugin marketplace add backnotprop/plannotator
/plugin install plannotator@plannotator
```
